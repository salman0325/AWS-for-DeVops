1️⃣ What is RDS?

👉 Answer:
Amazon Web Services RDS (Relational Database Service) is a managed service that helps you set up, operate, and scale a relational database easily.

2️⃣ Which databases does RDS support?

👉 Answer:
RDS supports:

MySQL
PostgreSQL
MariaDB
Oracle
SQL Server
Amazon Aurora
3️⃣ What is Multi-AZ in RDS?

👉 Answer:
Multi-AZ means:

A standby copy of your database is created in another Availability Zone
If the primary database fails, automatic failover happens
👉 Used for high availability
4️⃣ What is a Read Replica?

👉 Answer:

A Read Replica is a read-only copy of the main database
It is used to handle read traffic and improve performance
5️⃣ What is Automated Backup in RDS?

👉 Answer:

RDS automatically takes daily backups
You can restore the database to a specific point in time
👉 This is called Point-in-Time Recovery
